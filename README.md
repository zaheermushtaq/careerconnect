# CareerConnect

A LinkedIn-inspired job portal built with the MERN stack (MongoDB, Express, React, Node.js).

## Status
🚧 Project in progress — currently setting up development environment.

## Tech Stack
- Frontend: React (Vite), Tailwind CSS, shadcn/ui, Framer Motion
- Backend: Node.js, Express.js, MongoDB, Mongoose, JWT
- Extras: Cloudinary, Socket.io
